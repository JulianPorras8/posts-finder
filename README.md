# typescript-react-nextjs-redux-starter

Minimal starter with typescript, react, nextjs, redux, redux-saga, emotion

### INSTALL

<pre>
<code>
#install node_modules
#server
cd typescript-react-nextjs-redux-starter/server/
npm install

#client
cd typescript-react-nextjs-redux-starter/client/
npm install
</code>
</pre>

### DEV MODE

<pre>
<code>
#run app on dev mode
#server
cd typescript-react-nextjs-redux-starter/server/
npm run dev

#client
cd typescript-react-nextjs-redux-starter/client/
npm run dev
</code>
</pre>

### BUILD AND START

<pre>
<code>
#app build and start
#server
cd typescript-react-nextjs-redux-starter/server/
npm start

#client
cd typescript-react-nextjs-redux-starter/client/
npm start
</code>
</pre>
